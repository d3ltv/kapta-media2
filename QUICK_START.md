# 🚀 Quick Start - Optimisations Performance

## ⚡ En 3 Commandes

```bash
# 1. Installer
cd frontend && npm install && cd ..

# 2. Build
cd frontend && npm run build && cd ..

# 3. Déployer
vercel --prod
```

## ✅ Vérification Rapide

1. **Ouvrir le site déployé**
2. **F12 → Lighthouse → Performance**
3. **Vérifier Score > 75** ✨

## 🎯 Pour aller à 85-90

**Optimiser le logo** (2 minutes) :
1. https://squoosh.app/
2. Upload `frontend/public/logo.webp`
3. Quality: 85, Width: 128
4. Download → remplacer
5. Rebuild + redéployer
