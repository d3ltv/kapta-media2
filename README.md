# KAPTA Media - Site Web

Agence de marketing local spécialisée dans l'optimisation Google Maps et la création de contenu vidéo professionnel à Tours.

## 🚀 Démarrage rapide

```bash
# Installation
cd frontend
yarn install

# Développement
yarn start

# Build production
yarn build

# Déploiement
vercel --prod
```
